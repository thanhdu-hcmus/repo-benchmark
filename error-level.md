Easy (Linter Errors)
- Unused variable
- Unused import
- Magic number
- Naming convention violation
- Missing null check
- Empty catch block
- Dead code / unreachable code
- Long method
- Duplicate code
- Missing braces in conditional
- Redundant condition
- Trailing whitespace

Medium (Business Logic Errors)
- Off-by-one error
- Incorrect condition logic
- Incorrect discount calculation
- Integer division precision loss
- Missing null handling in logic
- Incorrect date comparison
- Missing validation
- Incorrect loop boundary
- Improper error handling
- Incorrect state update
- Incorrect authentication logic

Hard (Architecture / Design Errors)
- SQL injection vulnerability
- Hardcoded secrets / credentials
- Race condition (thread safety issue)
- Resource leak (unclosed resource)
- N+1 query problem
- Tight coupling between components
- God class (too many responsibilities)
- Violation of Single Responsibility Principle
- Missing dependency injection
- Transaction management issue
- Improper exception propagation
- Circular dependency
